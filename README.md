AI-Based Digital Forensics for Crime Scene Investigations

Project Overview
This project is a Django-based web application that uses Artificial Intelligence techniques to support digital crime scene investigations. It provides user and admin modules, dataset visualization, AI model execution, and image testing for forensic analysis.

 Features
- User Registration and Login
- Admin Login and User Management
- Crime Scene Dataset Viewer
- AI-Based Crime Scene Analysis
- Image Upload and Prediction
- AI Model Performance Report
- Secure Authentication

 Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- TensorFlow
- NumPy
- Pandas
- OpenCV
- Matplotlib
- SQLite
